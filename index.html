Habilidades tecnológicas

+

HTML 5

Sua sigla vem do inglês ‘HyperText Markup Language’. Traduzido para o espanhol Significa 'Linguagem de marcação e hipertexto'. Esta linguagem de marcação nos permite criar a estrutura (ou corpo) de um site usando tags. O número cinco (5) faz referência à versão onde você está atualmente. Todos os títulos, parágrafos, formulários, vídeos, entre muitas outras coisas, nós os temos graças a esta linguagem maravilhosa.

+

CSS

CSS é uma linguagem de estilo. Sua sigla também tem o significado de 'Cascading Folha de estilo', Sua tradução é 'Folhas estilo cascata'. Certamente você já sabe o que ele quer dizer número três (3), VERDADEIRO? Exato! Sua versão atual. Esta linguagem de estilo nos permite... bem, dar estilos ao corpo do nosso site rede. É a composição do HTML. Todo o design de um site está diretamente relacionado esta linguagem.

+

JavaScript

Abençoado JavaScript! É uma linguagem de programação que nos permite adicionar, excluir, aplicar animações ou efeitos aos componentes do nosso site. É a ferramenta perfeita para fazer sites dinâmicos e muito atraente graças ao CSS e HTML. Com o tempo, evoluiu de exponencialmente e No momento é absolutamente necessário saber se você quer ser um desenvolvedor web.

+

Sql Server

O SQL Server é um sistema de gerenciamento de banco de dados relacional (RDBMS) desenvolvido pela Microsoft. Ele é amplamente utilizado para armazenar, organizar e gerenciar dados em aplicações de diversos tipos, especialmente em ambientes corporativos. O SQL Server utiliza a linguagem T-SQL (Transact-SQL), uma extensão do SQL padrão, que adiciona funcionalidades adicionais, como transações, manipulação de erros e controles de fluxo.

+

Ruby

Ruby é uma linguagem de programação de alto nível, orientada a objetos, projetada para ser simples e produtiva. Criada por Yukihiro "Matz" Matsumoto em 1995, Ruby combina a simplicidade e a elegância na sintaxe com poderosos recursos de programação.

+

Neo4J

Neo4j é um sistema de gerenciamento de banco de dados orientado a grafos, projetado para armazenar, manipular e consultar dados estruturados como grafos. Ao contrário de bancos de dados relacionais que utilizam tabelas e chaves primárias/estrangeiras, o Neo4j armazena dados em nós, relacionamentos e propriedades, o que o torna especialmente adequado para aplicações que envolvem redes complexas, como redes sociais, recomendações, gestão de fraudes e muito mais.
